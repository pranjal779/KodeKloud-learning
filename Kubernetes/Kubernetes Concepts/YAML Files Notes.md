# YAML

## Key Value Pair
```yaml
Fruit: Apple
Vegetable: Carrot
Liquid: Water
Meat: Chicken
```

## Array/List
```yaml
Fruits:
 - Orange
 - Apple
 - Banana

Vegetables:
 - Carrot
 - Cauliflower
 - Tomato
```

## Dictionary/Map
```yaml
Banana:
  Calories: 105
  Fat: 04 g
  Carbs: 27 g
Grapes:
  Calories: 62
  Fat: 0.3 g
  Carbs: 16 g
```
