# Day 13: IPtables Installation And Configuration

<img width="970" height="881" alt="image" src="https://github.com/user-attachments/assets/45eb4cb5-939e-44bb-b4c4-2a78e50730a8" />

We have one of our websites up and running on our Nautilus infrastructure in Stratos DC. 
Our security team has raised a concern that right now Apache’s port i.e 6000 is open for all since there is no firewall installed on these hosts. 
So we have decided to add some security layer for these hosts and after discussions and recommendations we have come up with the following requirements:

1. Install iptables and all its dependencies on each app host.

2. Block incoming port 6000 on all apps for everyone except for LBR host.

3. Make sure the rules remain, even after system reboot.
