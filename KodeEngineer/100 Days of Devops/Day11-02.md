```txt
Tomcat Deployment Procedural Tasks
Step 1: Install and Configure Tomcat
Install Tomcat: On the App Server 2 (stapp02), install the Tomcat server package using the package manager (e.g., sudo yum install tomcat or sudo apt install tomcat9).
Configure Port: Edit the Tomcat configuration file, typically located at /etc/tomcat/server.xml, to change the default port. Locate the <Connector port="8080" ... /> line and change it to <Connector port="8883" ... />.
Restart Service: Restart the Tomcat service to apply the configuration changes: sudo systemctl restart tomcat.
Step 2: Deploy WAR File
Copy WAR File: From the Jump host (jumphost), copy the ROOT.war file from /tmp/ to the Tomcat webapps directory on App Server 2 (usually /var/lib/tomcat/webapps/ or /var/www/tomcat/webapps/) using scp.
scp /tmp/ROOT.war steve@stapp02:/tmp/
Move to Webapps: On App Server 2, move the file to the correct directory and ensure permissions are correct: sudo mv /tmp/ROOT.war /var/lib/tomcat/webapps/.
Step 3: Verify Deployment
Check Service: Ensure Tomcat is running and listening on the new port: netstat -tlpn | grep 8083.
Verify Webpage: Use curl to verify the application is accessible on the base URL:
curl http://stapp02:8083
Answer:
The Tomcat server is installed on App Server 2, configured to run on port 8883, and the ROOT.war file is deployed to serve the application directly on the base URL via http://stapp02:8083.
```
